# アプリケーション名
task_calendar

# アプリケーション概要
- トップページからカレンダーの閲覧
- ユーザー登録・ログイン後、マイページでの予定登録・閲覧
- 予定の編集・削除

# 利用方法
- ログイン後、NewTaskをクリックすると新しいタスクの登録ページに遷移し、タスクの登録が行える。
- 登録後、Backをクリックするとマイページに遷移し、タスクの一覧及びカレンダーから登録したタスクを確認できる。
- マイページではタスクの編集や削除を行える。

# 目指した課題解決

仕事も趣味も頑張りたいと思っていたり、
個人的に勉強して資格の取得を目指したりしているユーザーが、
自分でタスクの管理を行えるようなアプリケーション。
ポイントとしては、
カレンダーの日別のタスク表示だけでなく、
タスク一覧の表示を設けたことと、
タスクごとのステータスを設け、達成感を得られるように工夫した。
