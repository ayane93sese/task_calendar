# アプリケーション名
task_calendar

# アプリケーション概要
- トップページからカレンダーの閲覧
- ユーザー登録・ログイン後、マイページでの予定登録・閲覧
- 予定の編集・削除

# 目指した課題解決

仕事も趣味も頑張りたいと思っていたり、
個人的に勉強して資格の取得を目指したりしている人が、
自分でタスクの管理を行えるようなアプリケーション。
ポイントとしては、
カレンダーの日別のタスク表示だけでなく、
タスク一覧の表示を設けたことと、
タスクごとのステータスを設け、達成感を得られるように工夫した。

# 洗い出した要件
| 機能             | 目的   | 詳細    |
| ------------------ | ------ | ---------- |
| トップページ      | カレンダー表示 | ・サインアップ、ログインページへ遷移できるボタンがある・ユーザーがログイン中だとログアウトボタンが表示される・当月のカレンダーが表示されている |
| ユーザー登録機能   | ユーザーごとにタスクを管理するため | ・ユーザー名、Eメールアドレス、パスワードでユーザー登録ができる |
| ユーザーログイン機能 | ログインすることによりタスクの閲覧や登録を行えるようにするため | ・ユーザー登録後、登録したEメールアドレスとパスワードでログインできる |
| マイページ | タスクの閲覧、登録、編集、削除などを行う | ・ユーザーがログイン後、マイページに遷移する・タスクの一覧を表示する場所がある・当月のカレンダーが表示されている・カレンダーにはユーザーの当月の予定が表示されている |
| タスク新規登録機能 | タスクの新規登録を行う | ・タイトル、日付、時間、詳細、カテゴリー、ステータスで登録できる・テキストは無くても登録できる・登録ページには登録ボタンと戻るボタンがある・カテゴリーとステータスはアクティブハッシュを使う |
| タスク詳細表示機能 | タスクの詳細情報を閲覧する | ・マイページのタスクのリンクをクリックすると遷移できる・タスクごとに詳細ページを表示できる・変更ボタンと削除ボタンも一緒に表示される |
| タスク変更機能 | タスクの詳細情報を変更する | ・詳細ページからタスク変更ページへ遷移できる・タスク新規登録ページとほぼ同じフォーム |
| タスク削除機能 | 一度登録したタスクを削除する | ・マイページまたはタスク詳細ページからタスクを削除できる・削除完了するとマイページへ遷移する |

# テーブル設計

## users テーブル


| Column             | Type   | Options    |
| ------------------ | ------ | ---------- |
| username           | string | null:false |
| email              | string | null:false |
| encrypted_password | string | null:false |

### Association

- has_many :tasks

## tasks テーブル

| Column      | Type       | Options                      |
| ----------- | ---------- | ---------------------------- |
| title       | string     | null:false                   |
| start_time  | datetime   | null:false                   |
| text        | text       |                              |
| category_id | integer    | null:false                   |
| rank_id     | integer    | null:false                   |
| status_id   | integer    | null:false                   |
| user        | references | null:false, foreign_key:true |

### Association

- belongs_to :user
